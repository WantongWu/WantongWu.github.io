Blog developing details

Posted on 2020-10-29 Edited on 2020-11-05
This post records details of progress in building this blog.

I am a newbie in computer science, and every errors below could appear to a newbie who wants to build blog on his or her own. It could be helpful reference.

Note: Underlined errors is copied from terminal. Errors not underlined are summary of the situation.

2020

October

10.29

The personal blog was built.

NexT theme was utilized.

November

11.1

(err: ValidationError: “slug” is required!)

Any publishment of new post is thrown to error.

11.2

“ValidationError: ‘slug’ is required!“ Solved.

Due to an error tips found in JavaScript “this.name is required!”, which was thrown because of new posts’ title in config was automatically “title.md”.

Solution is to change title.md in slug form.

There is no need to add slug in the beginning of the markdown file.

Error: pictures could not be shown

Although format to insert pic was right, the pictures still couldn’t be shown.

11.3

“Error: pictures could not be shown” Solved.

Install hexo-asset-image plugin:

Make sure value of post_asset_folder in _config.yml is true

post_asset_folder: true

(Tips: space should be added after colon)

A new markdown file would be generated with a folder. They should have same name. Put intended pictures in folder and insert them in markdown file.

To insert pic, use

For example:

# record
